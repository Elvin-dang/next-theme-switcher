# Theme Switcher Documentation

Welcome to the documentation for the **Next.js Theme Switcher**!

## Getting Started

To use the theme switcher in your project, follow these steps:

1. **Install Dependencies**

```bash
npm install next-theme-switcher
```

2. **Add the ThemeProvider**

Wrap your application with the `ThemeProvider` in your `_app.js` or layout file:

```jsx
import { ThemeProvider } from "next-theme-switcher";

export default function App({ Component, pageProps }) {
  return (
    <ThemeProvider>
      <Component {...pageProps} />
    </ThemeProvider>
  );
}
```

3. **Add the Switcher Component**

Place the theme switcher button where you want users to toggle themes:

```jsx
import { ThemeSwitcher } from "next-theme-switcher";

export default function Header() {
  return (
    <header>
      <ThemeSwitcher />
    </header>
  );
}
```
